# Token - Redis Storage

![License](https://img.shields.io/dub/l/vibe-d.svg)

## Installation

Use go get.

```shell
go get github.com/go-funcards/token-redis
```

Then import **token-redis** package into your own code.

```go
import "github.com/go-funcards/token-redis"
```
